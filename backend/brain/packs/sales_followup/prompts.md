\# Sales Follow‑Up Brain — Prompt Structures



These prompt structures apply the Decision Rules and Guardrails

to generate professional, high‑quality follow‑up messages.



Each prompt assumes that context is provided from a real interaction.



\---



\## Prompt 1: First Follow‑Up After Meeting



Generate a professional follow‑up email after an initial sales meeting.



The message must:

\- Reference the specific meeting

\- Acknowledge one discussed topic

\- Contain a single, clear next step



Context to include:

\- Meeting type

\- Key discussion point

\- Desired next action



\---



\## Prompt 2: Follow‑Up After Demo or Presentation



Generate a follow‑up message after a product demo or presentation.



The message must:

\- Reference the demo explicitly

\- Reconnect to the prospect’s stated need

\- Request a concrete next step



Context to include:

\- Demo date

\- Prospect’s priority

\- Proposed action (call, meeting, confirmation)



\---



\## Prompt 3: Second Follow‑Up (No Response)



Generate a polite but more direct follow‑up when there was no response

to a previous message.



The message must:

\- Acknowledge the previous follow‑up

\- Reduce message length

\- Reinforce a single CTA



The tone must remain professional and confident.



\---



\## Prompt 4: Final Check‑In



Generate a final check‑in message indicating availability

without pressure or urgency.



The message must:

\- Respect the prospect’s time

\- Avoid guilt or escalation

\- Offer a simple path forward



Context to include:

\- Time since last interaction

\- Optional opt‑out acknowledgement
